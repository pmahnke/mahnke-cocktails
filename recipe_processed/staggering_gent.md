### Ingredients

| Amount | Ingredient              | Brand           |
| -----: | ----------------------- | --------------- |
|   2 oz / 50 ml | Barrel Strength Bourbon | Stagg Jr.       |
| 0.5 oz / 12.5 ml | Bénédictine             | Bénédictine DOM |
| 0.5 oz / 12.5 ml | Walnut Liqueur          | Nux Alpina      |

### Notes

- Method: Stirred
- Pour: Julep Strain
- Glassware: Low BallLarge Ice cube
- Garnish: None
- Special Prep: None
