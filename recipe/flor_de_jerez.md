### Ingredients

|  Amount | Ingredient             | Brand                      |
| ------: | ---------------------- | -------------------------- |
|  1.5 oz | Amontillado Sherry     |
|  0.5 oz | Jamaican Rum           |
| 0.25 oz | Apricot Liqueur |
| 0.75 oz | Fresh Lemon Juice      |
|  0.5 oz | Cane Syrup             |
|  1 dash | Bitters                | Angostura Aromatic Bitters |

### Notes

- Method: Shaken
- Pour: Double Strain
- Glassware: Coupe
- Garnish: None
- Special Prep: None
