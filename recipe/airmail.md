### Ingredients

| Amount | Ingredient                                    | Brand |
| -----: | --------------------------------------------- | ----- |
|   1 oz | Gold Rum                                      | Don Q |

| 0.5 oz | Fresh Lime Juice                              |       |
|   2 oz | Champagne                                     |       |

### Notes

- Method: Shaken
- Pour: Double Strain
- Glassware: Coupe
- Garnish: None
- Special Prep: Pour cocktail into glass and top it off with the champagne.
